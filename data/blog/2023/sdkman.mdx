SDKMAN은 Java, Kotlin, Groovy, Scala, Maven, Gradle, Ant, SBT, Spring Boot, Grails, Vert.x 등 JVM 기반 언어와 프레임워크를 관리해주는 툴입니다.

맥북에서 SDKMAN을 사용해 JAVA를 설치하고 관리하는 방법에 대해 알아보도록 하겠습니다.

## 설치
터미널에 아래 명령어를 입력하여 SDKMAN을 설치합니다.
```bash
curl -s "https://get.sdkman.io" | bash
```

## 사용법

### 설치 가능한 java 버전 확인
```bash
$ sdk list java
```

### java 설치
```bash
$ sdk install java
```

### 버전을 정의하여 java 설치
```bash
$ sdk install java 20.0.2-amzn
```

### 설치된 java 버전 확인
```bash
$ java -version
```

### 설치된 java 버전 변경
```bash
$ sdk use java 20.0.2-amzn
```

### 설치된 java 버전 삭제
```bash
$ sdk uninstall java 20.0.2-amzn
```

### java가 설치된 경로
```bash
{유저홈}/.sdkman/candidates/java/{설치된 버전}
```
